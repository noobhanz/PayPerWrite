# Error Codes (Canonical)

- Unauthorized: Caller lacks required signer or token ownership.
- InsufficientPayment: Buyer did not provide required amount.
- NotRentExempt: Account size/rent not satisfied.
- ArticleNotFound: PDA missing or closed.
- AlreadyPurchased: Receipt exists for (article, buyer) when SBT is required.
- TransferBlocked: Attempted to transfer SBT token.
