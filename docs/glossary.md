# Glossary

- Article: A content item listed for one-time purchase.
- Access Token: NFT or SBT minted to a buyer to prove access rights.
- Receipt: On-chain record of a paid purchase for analytics/refunds.
- Encrypted URI: Arweave/IPFS CID of encrypted content blob.
- Lit Conditions: Access control rules requiring token ownership to release the decryption key.
